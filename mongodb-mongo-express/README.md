# mongodb / mongo-express

## Defaults

See [.env](./.env) file for the defaults

## Start

```shell
docker-compose up -d
```

## Stop

```shell
docker-compose down
```

## Stop / Clean

```shell
docker-compose down -v
```

## Links

[mongo-express](http://localhost:8002/) Use `MONGODB_ROOT_USER` and `MONGODB_ROOT_PASSWORD` defined in the [.env](./.env) file to login
